# Adds iBAQ values based on LFQ values to SummarizedExperiment

Adds iBAQ values based on LFQ values to SummarizedExperiment

## Usage

``` r
add_maxLFQ_iBAQ(se)
```

## Arguments

- se:

  SummarizedExperiment object in prepare_se

## Value

A Summarized Experiment object with extra assay.

## Examples

``` r
if (FALSE) { # \dontrun{
se <- add_maxLFQ_iBAQ(se)
} # }
```
