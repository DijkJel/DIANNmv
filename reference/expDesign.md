# Experimental design for example data

Necessary for creating SummarizedExperiment object with MS data

## Usage

``` r
expDesign
```

## Format

### `expDesign`

A data frame with 6 rows and 3 columns:

- label:

  column name in report.pg_matrix specifying MS sample

- condition:

  The different experimental conditions

- replicate:

  The different replicates per condition
